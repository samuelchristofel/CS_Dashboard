# Development Tools: Vastel Helpdesk Application

> **Purpose**: High-level overview of the core technologies used in this project.

---

## Core Technology Stack

| Category | Tool | Description |
|----------|------|-------------|
| **Runtime** | Node.js | Server-side JavaScript runtime environment |
| **Language** | TypeScript | Strongly typed superset of JavaScript |
| **Framework** | Next.js | Full-stack React framework for web applications |
| **Library** | React | Library for building user interfaces |

---

## Database & Data Management

| Category | Tool | Description |
|----------|------|-------------|
| **Database** | MySQL | Relational database management system |
| **ORM** | Prisma | Tool for database access and schema management |

---

## Styling & UI

| Category | Tool | Description |
|----------|------|-------------|
| **CSS Framework** | Tailwind CSS | Utility-first CSS framework for rapid styling |
| **Notifications** | React Hot Toast | Lightweight notifications for user feedback |
| **Charts** | Recharts | Composable charting library for React |
