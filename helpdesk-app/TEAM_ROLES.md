# Team Roles: Vastel Helpdesk Application

> **Team Size**: 3 People

---

## Person 1: Backend Developer

**Focus**: Database, API, and server logic

| Responsibility | What Was Built |
|----------------|----------------|
| Database | MySQL + Prisma schema (Users, Tickets, Notes, Activities, Chat) |
| Authentication | Login API, credential validation |
| Ticket APIs | Create, read, update, delete tickets with filters |
| User APIs | User management for Admin |
| Stats/KPI APIs | Dashboard statistics, performance scoring |
| Chat APIs | Conversations and messaging |

---

## Person 2: Frontend Developer

**Focus**: React pages and components

| Responsibility | What Was Built |
|----------------|----------------|
| Login Page | Authentication form, session handling |
| Dashboards | 4 role-specific dashboards (Admin, Senior, Junior, IT) |
| Components | Cards, modals, dropdowns, buttons |
| Chat Widget | Floating draggable chat window |

---

## Person 3: Tester / QA

**Focus**: Quality assurance and testing

| Responsibility | What Was Tested |
|----------------|-----------------|
| API Testing | All endpoints return correct data |
| User Flows | Login, ticket actions, role permissions |
| Edge Cases | Empty states, validation errors |
| Bug Reporting | Document issues, verify fixes |

---

## Shared Work

| Area | All Contributed |
|------|-----------------|
| Project Setup | Next.js, folder structure, environment config |
| Documentation | README, credentials, deployment guides |
